# Multilingual Hate Speech Detection

## Overview

Developed a multilingual hate speech detection system using XLM-RoBERTa and Hugging Face Transformers to classify toxic and non-toxic text.

## Features

* Multilingual text classification
* Toxicity detection
* Transformer-based NLP model
* Model evaluation using Accuracy and F1 Score

## Technologies Used

* Python
* Hugging Face Transformers
* XLM-RoBERTa
* PyTorch
* Datasets Library
* Scikit-learn

## Dataset

TextDetox Multilingual Toxicity Dataset

## Skills Demonstrated

* Natural Language Processing (NLP)
* Deep Learning
* Text Classification
* Transformer Models
* Model Evaluation
